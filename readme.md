# ren-vm project

## How to run
- Clone repo
- Cd to repo
- Run `make && ./REN-VM`

## Instruction set
PSH (int): Pushes value to top of stack
ADD: Pops two values on the top of stack and adds them
POP: Pops top value on the stack
SET (reg) (int): Sets value of register
GET: Pushes value in register to stack
MOV - Not finished
PRN - Not finished
GOT - Not finished
HLT: Halts program
